# 中国劳动法 — 辞退谈判知识库

> 基于《中华人民共和国劳动法》《中华人民共和国劳动合同法》及司法解释
> 用途: 为辞退/离职谈判提供法律框架参考，辅助 HR 做出合规决策

---

## 1. 解除劳动合同的法定情形

### 一、协商解除（劳动合同法第36条）
- 用人单位与劳动者协商一致，可以解除劳动合同
- **谈判要点**: 协商解除是风险最低的方式，核心是补偿方案达成一致
- **注意**: 协商解除协议需书面形式，明确"双方再无争议"

### 二、劳动者过失性解除（第39条 — 无需支付补偿金）
| 情形 | 法条 | 关键证据要求 |
|------|------|-------------|
| 试用期不符合录用条件 | 39(1) | 需有明确的录用条件 + 考核记录 |
| 严重违反规章制度 | 39(2) | 制度需经民主程序 + 公示 + 违纪证据 |
| 严重失职/营私舞弊造成重大损害 | 39(3) | 需证明因果关系 + 损害程度 |
| 劳动者同时与其他单位建立劳动关系 | 39(4) | 对完成本单位工作造成严重影响且拒不改正 |
| 欺诈致合同无效 | 39(5) | 如学历造假、经历造假 |
| 被追究刑事责任 | 39(6) | 需生效判决 |

### 三、无过失性解除（第40条 — 需支付补偿金 + 提前30天通知）
| 情形 | 适用条件 | 补偿 |
|------|---------|------|
| 医疗期满不能从事原工作 | 医疗期计算：工龄 <10年 = 3个月/年 | N+1 |
| 不胜任工作，经培训调岗仍不胜任 | 需有绩效改进计划(PIP)证据 | N+1 |
| 客观情况重大变化致合同无法履行 | 如部门裁撤、业务调整 | N+1 |

### 四、经济性裁员（第41条）
- 条件: 裁减20人以上或占职工总数10%以上
- 程序: 提前30天向工会/全体职工说明 → 听取意见 → 报告劳动行政部门
- 优先留用: 长期合同、无固定期限合同、家庭无其他就业人员
- 补偿: N

## 2. 经济补偿金（N）计算

### 计算公式
```
经济补偿金 = 劳动者解除前12个月平均工资 × 本单位工作年限
```

### 工作年限计算规则
| 规则 | 说明 |
|------|------|
| 满1年算1个月 | 每满1年支付1个月工资 |
| 6个月以上不满1年 | 按1年计算（支付1个月） |
| 不满6个月 | 支付半个月工资 |

### 工资基数
- **月工资**: 解除前12个月的平均工资
- **上限**: 高于当地上年度职工月平均工资3倍的，按3倍计算，年限最高12年

### N+1 场景
- 适用第40条（医疗期满/不胜任/客观情况变化）
- "+1" = 额外支付1个月工资（代通知金），可替代30天提前通知
- **注意**: N+1 中的"+1"是额外1个月，不乘以年限

### 违法解除（第87条）— 2N
- 用人单位违法解除或终止合同 → 支付 **2倍** 经济补偿金
- 常见违法情形: 无正当理由辞退、程序不合法、证据不足

## 3. 辞退谈判策略框架

### 谈判准备清单
```
□ 确认解除类型: 协商解除 / 过失性 / 无过失性
□ 计算法定补偿: N / N+1 / 2N
□ 准备证据材料: 违纪记录、绩效数据、规章制度
□ 评估劳动者预期: 已知诉求、心理底线
□ 准备谈判方案: 最优/次优/底线
```

### 三类谈判场景

#### 场景A: 过失性辞退（有证据）
- **法律位置**: 法定0补偿
- **谈判策略**: 以协商解除为名义，给予有限补偿换取和平分手
- **话术框架**: "基于[具体行为/事实]，公司本可依据39条解除且无需补偿。但考虑到你的贡献，我们愿意以协商方式，给你[X个月]补偿作为过渡支持。"
- **风险**: 员工可能主张证据不足 → 提前评估证据链强度

#### 场景B: 绩效/能力原因（PIP后）
- **法律位置**: N+1
- **谈判策略**: 以N+1为锚，可适当上浮换取快速签署
- **话术框架**: "根据绩效改进计划和这[X个月]的考核记录，我们已经做了最大努力。公司愿意按N+1标准支付补偿，并额外提供[社保延续/推荐信/缓冲期]"
- **筹码**: 额外1-2个月 + 延长社保 + 正面离职证明

#### 场景C: 协商一致（无过失）
- **法律位置**: 双方自由协商
- **谈判策略**: 以N为起点，了解对方期望，寻找交集
- **话术框架**: "这次调整是公司战略决定，不是对你的否定。我们希望能以体面的方式结束这段合作。你看这样是否合理..."
- **筹码**: 补偿金 + 离职时间灵活性 + 推荐信 + 竞业限制补偿

### 谈判中的法律底线
| 严禁行为 | 后果 |
|---------|------|
| 威胁、恐吓逼迫签字 | 协议无效 + 可能构成强迫 |
| 虚假承诺（口头说有补偿，协议不写） | 协议可撤销 |
| 不给员工看协议直接让签 | 可能被认定欺诈 |
| 在孕期/产期辞退（无正当理由） | 违法解除 + 2N |

## 4. 特殊情况

### 三期女职工（孕期、产期、哺乳期）
- **原则**: 不得依据第40条、第41条解除
- **例外**: 仅可依据第39条（过失性）解除
- **谈判**: 需极其谨慎，建议协商为主

### 医疗期员工
- **医疗期**: 3个月~24个月（依据工龄）
- **解除条件**: 医疗期满 + 不能从事原工作 + 不能从事另行安排的工作
- **补偿**: N+1 + 医疗补助费（不低于6个月工资，重病加倍）

### 工伤员工
- 1-4级: 不得解除
- 5-6级: 需本人提出方可解除
- 7-10级: 可依法解除，需支付一次性工伤医疗补助金 + 伤残就业补助金

## 5. 竞业限制与保密

### 竞业限制
- **最长**: 不超过2年
- **补偿**: 不低于劳动合同终止前12个月平均工资的30%，且不低于最低工资标准
- **适用范围**: 高级管理人员、高级技术人员、其他负有保密义务的人员
- **谈判要点**: 辞退时可协商是否保留竞业限制（如不保留可节省补偿支出）

### 保密义务
- 即使未签保密协议，对工作中知悉的商业秘密仍负保密义务
- 离职时可签署《离职保密承诺书》作为补充

## 6. 离职协议关键条款

### 必备条款
```
1. 解除日期（明确最后工作日）
2. 补偿金额及支付时间
3. 工资结算截止日
4. 社保公积金缴纳至何时
5. 竞业限制是否保留
6. "双方再无争议"条款（这是关键保护）
```

### 禁止条款
- "员工放弃追究法律责任"（可能被认定无效）
- "补偿包含所有款项但不列明细"（需列明各项补偿）

### 注意事项
- 协议需双方签字，公司盖章
- 支付补偿金需在解除后15日内完成
- 离职证明需在解除时出具
- 档案和社保转移需在15日内办理

## 7. 辞退谈判心理策略

### 情绪管理
- 员工可能经历: 震惊→否认→愤怒→讨价还价→接受
- HR应: 保持冷静、给予尊重、不争论、聚焦方案

### 关键原则
- **以法为基**: 所有方案必须合法合规，不要给"法外承诺"
- **以人为本**: 承认对方的贡献和感受，体面分手
- **利益导向**: 把对话从"谁对谁错"转向"双方如何体面结束"
- **底线思维**: 明确公司底线，不因对方情绪而突破法律边界

### 谈判陷阱
- 员工录音 → 假设每场对话都被录音，措辞谨慎
- 员工"我还有证据" → 不要被唬住，要求对方出示
- 员工"我要找劳动仲裁" → 冷静回应：这是你的权利，我们尊重。但先看看方案是否合理
