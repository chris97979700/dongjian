# 沟通经典 & 分析框架 — 补充知识库

> 以下框架萃取自经典著作，用于增强洞见对候选人语言、逻辑和思维模式的解析能力。

---

## 1. 非暴力沟通 (Nonviolent Communication — Marshall Rosenberg)

### 四步框架 (OFNR)
| 步骤 | 含义 | 面试中的应用 |
|------|------|-------------|
| 观察 (Observation) | 区分事实与评价 | 候选人是说"他迟到了3次"(事实)还是"他不靠谱"(评价)？事实型 = 高尽责 |
| 感受 (Feeling) | 识别真实情绪 | 候选人能准确命名情绪("焦虑""沮丧")还是笼统("不爽")？精确 = 高情绪智力 |
| 需要 (Need) | 情绪背后的需求 | "我需要被认可" vs "他们不尊重我" → 前者内控，后者外控 |
| 请求 (Request) | 清晰可执行的请求 | 模糊请求("希望更好")→ 低行动力；具体请求 → 高执行力 |

### 分析信号
- **暴力的语言模式**: 指责("你总是...")、比较("别人都...")、强求("必须/应该")、贴标签 → 暗黑三角↑
- **非暴力标志**: 为自己的感受负责("我感到...因为我需要...") → 高宜人性 + 内控归因
- **同理心表达**: 能否复述对方观点再回应？→ 倾听能力 + 宜人性

## 2. 关键对话 (Crucial Conversations — Patterson, Grenny, McMillan, Switzler)

### 关键对话三要素
高风险 + 观点分歧 + 强烈情绪 = 此时最能暴露一个人的真实沟通模式

### STATE 对话模型
| 步骤 | 操作 | 候选人信号 |
|------|------|-----------|
| Share facts | 先摆事实 | 能否区分事实与判断？→ 理性 vs 情绪化 |
| Tell story | 说出你的故事 | 自我叙事是建设性("我想可能是...")还是攻击性("他就是...")？ |
| Ask for others' path | 邀请对方分享 | 是否主动询问面试官看法？→ 宜人性 + 开放性 |
| Talk tentatively | 试探性表达 | "也许""可能"的使用 = 认知谦逊 vs "一定""绝对" = 僵化思维 |
| Encourage testing | 鼓励验证 | 是否欢迎不同观点？→ 神经质(低) |

### 安全信号
- **沉默或暴力**: 面对压力问题时回避/攻击 → 如果在面试中都出现，工作中更甚
- **对比法**: "我不希望...但是我想..." → 高情商的冲突处理
- **CPR分析**: Content(事件) → Pattern(模式) → Relationship(关系) — 候选人如何升级问题层级？

## 3. 沟通的艺术 (Looking Out, Looking In — Adler & Proctor)

### 自我概念
- **自我应验预言**: "我肯定做不好" → 低自我效能，可能低尽责
- **认同管理**: 面试中刻意呈现的形象 vs 真实自我 → 差距越大 = 马基雅维利↑

### 知觉核验 (Perception Checking)
1. 描述你注意到的行为
2. 提出至少两种可能的解释
3. 请求澄清

候选人能否做到？→ 反映归因风格和认知复杂度

### 倾听层次
| 层次 | 行为 | 信号 |
|------|------|------|
| 伪倾听 | 点头但不吸收 | 回应与问题脱节 |
| 选择性倾听 | 只听自己关心的 | 对薪资信号敏感，对方向问题敷衍 |
| 深度倾听 | 复述+追问 | 高宜人 + 高开放性 |

## 4. 学会提问 (Asking the Right Questions — Browne & Keeley)

### 批判性思维检查表（面试版）

| 检查点 | 对候选人的分析 |
|--------|---------------|
| 结论是什么？ | 候选人表达的是事实、观点还是情绪？ |
| 理由是什么？ | "我离职是因为..." → 理由充分 vs 模糊 |
| 哪些词有歧义？ | "成长空间""重视"→ 追问具体含义，测试清晰度 |
| 价值观假设是什么？ | 候选人认为什么重要(钱/成长/稳定)？ |
| 描述性假设是什么？ | "跳槽多=不忠诚" → 检查是否隐含不合理的假设 |
| 推理中有谬误吗？ | 稻草人、虚假两难、诉诸情感、以偏概全 |

### 常见谬误识别
- **虚假两难**: "要么给我80万，要么就是不重视" → 框架绑架
- **诉诸流行**: "大家都这样做" → 缺乏独立思考
- **滑坡谬误**: "如果这次不涨，以后都不会涨了" → 灾难化思维
- **人身攻击/诉诸动机**: "HR都是帮公司压价的" → 防御性/低宜人

## 5. 金字塔原理 (The Pyramid Principle — Barbara Minto)

### 核心：结论先行，以上统下

### SCQA 框架
- **Situation** (情境): 描述已知事实
- **Complication** (冲突): 指出问题或挑战
- **Question** (问题): 引出需要回答的问题
- **Answer** (答案): 给出结论

候选人能否自然地使用 SCQA 结构表达？→ 思考力+会抓重点

### MECE 原则 (Mutually Exclusive, Collectively Exhaustive)
- 分析问题时能否做到不重叠、不遗漏？
- "我觉得有三个原因，第一...第二...第三..." → 结构思考力
- "就是感觉不太对" → 缺乏结构化思维

### 演绎 vs 归纳
- **演绎**: 大前提→小前提→结论 → 逻辑严密型
- **归纳**: 现象→共性→结论 → 模式识别型
- 候选人偏哪种？→ 匹配岗位需求

## 6. 毛选 — 矛盾论·实践论·辩证法

### 矛盾论 (On Contradiction)
| 概念 | 应用于人才分析 |
|------|---------------|
| **主要矛盾 vs 次要矛盾** | 候选人最核心的诉求是什么？（钱？成长？安全？）• 不要被次要信息分散 |
| **矛盾的主要方面** | 在候选人的离职/入职决策中，哪个因素是主导的？ |
| **矛盾的同一性和斗争性** | 候选人的"想要高薪"和"想要稳定"可能矛盾，如何统一？ |
| **对抗性 vs 非对抗性矛盾** | 薪资分歧是非对抗性（可谈），价值观冲突是对抗性（应放弃）|

### 实践论 (On Practice)
- **认识从实践始**: 候选人说的 ≠ 候选人做的，寻找行为证据而非口头承诺
- **感性认识→理性认识**: 从候选人的具体故事/经历中抽象出行为模式
- **理论要回到实践中检验**: 任何分析结论需要在后续面试中验证

### 调查研究
> "没有调查，就没有发言权" — 毛泽东

- 对候选人的判断必须基于充足的对话证据
- 单一信号不足以定论，需要多方印证
- "你看到的是现象还是本质？"

### 分析方法论
| 方法 | 应用 |
|------|------|
| **一分为二** | 每个候选人都有优点和风险，不极端化 |
| **透过现象看本质** | "他要高薪"是现象，本质可能是"需要价值认可" |
| **具体问题具体分析** | 不套模板，每个候选人的背景、行业、阶段都不同 |
| **抓住主要矛盾** | 面试中识别最关键的信号，不被噪音干扰 |
| **发展的观点** | 候选人现在的状态不等于未来的表现，动态评估 |
| **量变到质变** | 多个小信号累积才能形成可靠判断，不因一个信号过度反应 |

### 语言风格分析（毛选特色）
- **务实 vs 空谈**: "我做了什么" vs "我认为应该..."
- **群众路线**: "我们团队一起..." vs "我一个人搞定..." → 归属感 vs 自恋
- **批评与自我批评**: 能否客观评价自己的不足？→ 高宜人 + 内控归因
- **实事求是**: 用数据和事实说话 → 高尽责 + 高思考力

## 综合应用指南

### 面试对话分析增强矩阵

| 流派 | 分析维度 | 关键信号 |
|------|---------|---------|
| 非暴力沟通 | 语言暴力指数 | 指责/标签/强求 vs 观察/感受/需要 |
| 关键对话 | 压力对话能力 | STATE模式、安全感维护 |
| 沟通的艺术 | 自我概念 | 认同管理差距、倾听层次 |
| 学会提问 | 批判思维 | 结论-理由链、谬误识别 |
| 金字塔原理 | 思维结构 | SCQA结构、MECE、演绎/归纳 |
| 毛选 | 辩证分析 | 主要矛盾、实事求是、本质穿透 |

### 注意事项
- 这些框架用于增强分析维度，不替代原有心理学框架
- 候选人缺乏某些沟通技巧不等于人格缺陷（可能是训练不足）
- 面试场景中的沟通表现与工作中的沟通表现可能不同
- 结合文化背景：中文高语境文化下，某些"直接"或"间接"有不同的含义
