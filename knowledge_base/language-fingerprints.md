# 语言指纹分析 — 话语分析 & 语言心理学

> 基于 Pennebaker (LIWC), Vrij (欺骗检测), Adams (SCAN) 等研究
> 用途: 从候选人/员工对话记录中提取语言行为特征，推断心理状态与风险

---

## 1. 人称代词分析 (Pronoun Analysis)

### 第一人称单数 "我" vs 复数 "我们"

| 模式 | 含义 | 风险关联 |
|------|------|----------|
| 高频率 "我" | 自我关注、独立性、可能焦虑或抑郁 | 团队协作风险 ↑ |
| 低频率 "我" | 回避责任、可能隐瞒个人行为 | 诚信风险 ↑（尤其在需要承担责任的问题上） |
| 高频率 "我们" | 团队归属感、集体主义 | 适配风险 ↓ |
| 低频率 "我们" | 疏离感、不认同团队 | 离职风险 ↑ |
| "我"→"我们"突然切换 | 责任转移信号 | 诚信风险 ↑ |

### 第二人称 "你"

| 模式 | 含义 |
|------|------|
| 高频 "你" | 攻击性、指责倾向、防御心理 |
| 泛指 "你"（如"你知道"） | 寻求认同、降低陈述确定感 |

### 第三人称 "他/她/他们"

| 模式 | 含义 |
|------|------|
| 高频第三人称 | 心理距离、将问题外化 |
| 具体人名 vs 模糊指代 | 前者关联性强，后者可能回避 |

### 无人称（零人称）

| 模式 | 含义 | 风险 |
|------|------|------|
| "需要做" vs "我会做" | 回避承诺 | 诚信风险 ↑ |
| 被动语态高频 | 回避责任承担 | 诚信风险 ↑ |

## 2. 认知过程词汇 (Cognitive Process Words)

> LIWC 分类体系核心维度

### 因果词汇 (Causal Words)
`因为 所以 因此 导致 由于 结果 影响`

| 模式 | 含义 |
|------|------|
| 高频因果词 | 理性思维强，善于分析——但也可能过度合理化 |
| 低频因果词 | 可能回避解释，或思维跳跃 |
| 外部归因因果词（"因为市场/因为领导"） | 外控型，推卸责任倾向 |
| 内部归因因果词（"因为我决定/我选择"） | 内控型，责任感强 |

### 洞察词汇 (Insight Words)
`理解 意识到 发现 明白 知道 想通`

| 模式 | 含义 |
|------|------|
| 高频洞察词 | 反思能力强，成长潜力高 |
| 低频洞察词 | 可能自我认知不足 |
| 面试中过度使用 | 可能是"表演性反思" |

### 不确定性标记 (Tentative Words)
`可能 也许 大概 好像 似乎 差不多 不一定`

| 语境 | 含义 |
|------|------|
| 描述客观事实时高频使用 | 信息不确定，或有意模糊 |
| 描述自身经历时高频使用 | 可能记忆不真实 |
| 适度使用 | 思维严谨的正常表现 |

### 确定性标记 (Certainty Words)
`一定 肯定 绝对 毫无疑问 从不 总是`

| 模式 | 含义 |
|------|------|
| 过度使用绝对词 | 可信度下降（真相通常有细微差别） |
| 描述负面事件使用绝对词 | 情绪化表达，稳定性风险 ↑ |

## 3. 情感词汇 (Affective Words)

### 积极情感词
`开心 满意 喜欢 热爱 感恩 期待`

- 适度使用：正常
- 过度使用：可能掩盖负面情绪、或表演性积极
- 对当前工作的积极词密度：预测工作满意度

### 负面情感词
`焦虑 担心 不满 生气 失望 厌倦`

- 高频负面词：情绪稳定性风险 ↑
- 对"过去经历"的高频负面词：归因外化
- 对"未来"的高频负面词：离职风险 ↑

### 情感极性突变
- 从积极突然转消极（或反之）：情绪不稳定信号
- 特定话题触发情绪变化：识别敏感点

## 4. 语言复杂度指标

### 词汇多样性 (TTR / 类符-形符比)
| 模式 | 含义 |
|------|------|
| 高词汇多样性 | 认知复杂度高，但可能是刻意准备 |
| 低词汇多样性 | 可能是真实自发，但也可能语言能力不足 |
| 不同话题间TTR变化大 | 对特定话题感到困难/压力 |

### 句子长度
| 模式 | 含义 |
|------|------|
| 句子突然变短 | 回避、紧张、或准备不充分 |
| 句子过长且嵌套 | 可能在构建复杂叙事（注意核查一致性） |
| 正常波动 | 自然对话特征 |

### 停顿标记
| 标记 | 含义 |
|------|------|
| 填充停顿 "嗯...那个..." | 认知负荷增加——可能是回忆真实信息，也可能是构建虚假信息 |
| 无声停顿（可标注） | 需要结合上下文判断 |
| 重复 "我是说...我是说..." | 紧张或自我修正 |

## 5. 特定语言模式（欺骗检测信号）

> 基于 Vrij (2008), DePaulo (2003) 欺骗检测元分析

### 高可信信号（出现时可信度上升）
- 自发纠正（"不对，等一下，是周三不是周四"）
- 承认记忆不确定（"具体数字我记不太清了"）
- 包含可验证细节（具体时间、地点、人名）
- 情感与内容一致
- 提供不利己信息

### 低可信信号（出现时需警惕）
- 过度绝对化（"我百分之百确定"）
- 回避直接回答，迂回表达
- 过多修饰性细节（时间填充策略）
- 按时间顺序严格叙述（缺乏跳跃，像背稿）
- 过度强调诚实（"说真的""老实说""坦白讲"）
- 情感与内容不一致（描述愤怒时平静，描述快乐时平淡）

### 陈述结构信号
| 模式 | 信号 |
|------|------|
| 结构过于整洁 | 可能是编造的（真实回忆通常混乱） |
| 缺少感知信息（视觉、听觉、嗅觉细节） | 可能不是亲身经历 |
| 过度强调"不记得" | 选择性遗忘关键信息 |

## 6. 话题回避与转移

### 回避策略识别
- **问题反问**："你为什么会这么想？"
- **话题劫持**：用无关细节转移注意力
- **最小化**："就那样吧""没什么特别的"
- **过度概括**："基本上就是这些""大概就是这样"
- **幽默化解**：用玩笑避开严肃话题

### 敏感度信号
- 某个话题出现上述回避策略密集 → 标记为高敏感区
- 需要后续追问深入探查

---

## 7. 文化注意事项

- 高语境文化（中日韩）：间接表达是正常沟通方式，未必是回避
- 低语境文化（北美/北欧）：直接表达是常态，婉转表达更需注意
- 人称代词使用高度受语言影响（英语必用 I/you，中文可省略主语）
- 集体主义文化中"我们"高频是正常的，不宜直接判定为回避个人责任
