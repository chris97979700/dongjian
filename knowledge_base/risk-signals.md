# 风险信号识别 — 员工离职 & 诚信 & 稳定性

> 综合组织行为学、人力资源管理、犯罪心理学研究成果
> 用途: 从候选人/员工沟通中识别潜在风险信号，进行风险评级

---

## 1. 离职风险信号 — 基于离职前兆模型

> 参考: Mobley 离职过程模型, March & Simon 参与模型, Mitchell & Lee 工作嵌入理论

### 1.1 心理撤出信号 (Psychological Withdrawal)

| 阶段 | 信号 | 严重度 |
|------|------|--------|
| 认知撤出 | "最近在想自己的职业方向" | ⚠️ 早期 |
| 情感撤出 | "对公司没什么感觉了""就那样吧" | ⚠️ 中期 |
| 行为撤出 | 减少额外投入，只做分内事 | ⚠️ 中后 |
| 主动撤出 | 提到面试、新机会、行业动态 | 🚨 晚期 |

### 1.2 语言层面的离职信号

**时间导向变化:**
| 模式 | 含义 |
|------|------|
| 从"我们明年..."变为"公司明年..." | 心理距离拉大 |
| 不再使用将来时讨论团队 | 即将离开 |
| 对比性语言增多（"这里 vs 外面"） | 在评估其他选择 |
| 使用"如果留在这里"等条件句 | 离职可能性 ↑ |

**情感词汇变化:**
- 积极情感词汇关于公司的频率 ↓
- 中性/冷淡词汇替代曾经的热情表达
- 提到公司的语气从"我们公司"变为"这家公司"或"他们"

**投入度语言:**
| 信号 | 说明 |
|------|------|
| 从"我可以牵头这个"变为"你安排就行" | 主动→被动 |
| 对中长期项目的回避 | 不打算长期参与 |
| 学习/成长相关话题消失 | 不再为未来投资 |

### 1.3 离职风险评分矩阵

| 风险等级 | 信号组合 | 预估窗口 |
|----------|----------|----------|
| 🟢 低 | 无上述信号，积极未来导向 | 12个月+ |
| 🟡 中低 | 偶尔认知撤出，情感正常 | 6-12个月 |
| 🟠 中高 | 情感撤出 + 语言距离，投入下降 | 3-6个月 |
| 🔴 高 | 主动撤出信号 + 行为变化 | 0-3个月 |

## 2. 诚信风险信号

### 2.1 陈述一致性分析

| 检查维度 | 可信信号 | 风险信号 |
|----------|----------|----------|
| 时间线 | 自然跳跃，有回溯修正 | 过于完美，无修正 |
| 细节密度 | 核心细节多，外围细节少 | 外围细节异常多（填充） |
| 情感-内容 | 匹配 | 不匹配 |
| 跨次描述 | 核心一致，外围有变化 | 核心内容变化 |
| 可验证性 | 主动提供可验证信息 | 回避可验证细节 |

### 2.2 诚信风险语言标志

**高风险词汇模式:**
- "老实说""坦白讲""说真的" → 在非必要语境下过度出现
- "据我所知""在我的记忆里" → 为后续不一致留退路
- 过度使用宗教/道德词汇进行自我背书（"我对天发誓"）
- "按理说""原则上" → 规避具体责任

**回避模式:**
| 策略 | 示例 | 风险 |
|------|------|------|
| 模糊化 | "可能吧""差不多""记不太清" | ⚠️ 中 |
| 最小化 | "就是一点小问题""没什么大不了的" | ⚠️ 中高 |
| 合理化 | "在当时那种情况下，任何人都会..." | ⚠️ 中 |
| 反击 | "你为什么这么不信任我？" | ⚠️ 高 |
| 选择性遗忘 | 关键信息"不记得"，但其他细节清晰 | 🚨 高 |

### 2.3 诚信风险评级

| 等级 | 描述 | 建议 |
|------|------|------|
| 🟢 低风险 | 陈述自然，细节充足，情感一致 | 正常推进 |
| 🟡 注意 | 个别回避/模糊信号 | 针对性追问 |
| 🟠 关注 | 多重回避 + 不合理遗漏 | 深入调查 + 背景核实 |
| 🔴 高风险 | 严重不一致 + 核心回避 | 暂停推进，全面核查 |

## 3. 情绪稳定性风险

### 3.1 情绪调节能力指标

| 维度 | 健康标志 | 风险信号 |
|------|----------|----------|
| 情绪词汇多样性 | 丰富且适度 | 极端词密集（灾难化/理想化） |
| 压力叙事 | 有应对策略，有反思 | 只有受害叙事 |
| 人际冲突叙事 | 双方视角，承认己过 | 单方指责，完美受害者 |
| 情绪恢复叙事 | "之后我调整了" | "一直影响我到现在"（未解决）|

### 3.2 防御机制语言分析

> 基于 Vaillant 防御机制层级

| 防御层级 | 语言表现 | 心理健康 |
|----------|----------|----------|
| **成熟防御** (升华/幽默/利他) | "把精力投入到学习中""自嘲" | 🟢 健康 |
| **神经症防御** (理智化/压抑/置换) | "理性分析但回避情感" | 🟡 一般 |
| **不成熟防御** (投射/被动攻击/诉诸行动) | "都是他们的问题""摆烂" | 🟠 关注 |
| **精神病性防御** (否认/扭曲) | "根本没有这回事"（证据明确时） | 🔴 风险 |

### 3.3 情绪稳定性评级

| 等级 | 标志 |
|------|------|
| 🟢 稳定 | 情绪表达适当，有压力应对资源，使用成熟防御 |
| 🟡 轻度波动 | 偶有情绪化反应，能自我调节 |
| 🟠 中度风险 | 持续负面情绪，防御不成熟，需要关注 |
| 🔴 高风险 | 情绪严重不稳定，可能影响工作安全 |

## 4. 权力动机与人际风险

### 4.1 权力动机类型

| 类型 | 特征 | 风险 |
|------|------|------|
| **社会化权力** | "帮助团队成长""赋能他人" | 🟢 低 |
| **个人化权力** | "我要说了算""必须听我的" | 🟠 中高 |
| **补偿性权力** | 通过支配他人填补自卑 | 🔴 高 |

### 4.2 人际控制模式

| 控制类型 | 语言信号 | 风险 |
|----------|----------|------|
| 信息控制 | "这个你不需要知道" | ⚠️ 中 |
| 关系控制 | "你要站我这边" | ⚠️ 中高 |
| 情感控制 | "你这样让我很失望" | ⚠️ 高 |
| 现实控制 | 否认他人感知（煤气灯） | 🚨 严重 |

### 4.3 团队适配风险信号

| 信号 | 风险 |
|------|------|
| 过往经历中频繁的人际冲突 | ⚠️ 中高 |
| 总是"别人有问题"的归因模式 | ⚠️ 高 |
| 将合作描述为零和博弈 | ⚠️ 中高 |
| 对同事/下属的贬损性描述 | ⚠️ 高 |
| 缺乏跨部门合作经验的语言证据 | ⚠️ 中 |

```
├──────────────────┼──────┼───────────┼──────────────────────┤

│ 团队适配风险        │ 🟢  │ 中 (70%)  │ 合作导向语言           │
│ 权力动机风险        │ 🟢  │ 中 (55%)  │ 无明显个人化权力信号     │

└──────────────────┴──────┴───────────┴──────────────────────┘
```
