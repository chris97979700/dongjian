# 行为模式分析 — 人格 & 动机理论

> 基于大五人格 (Costa & McCrae), 暗黑三角 (Paulhus & Williams), 自我决定理论 (Deci & Ryan), 归因理论 (Weiner)
> 用途: 从对话中推断候选人/员工的人格特质和行为倾向

---

## 1. 大五人格 (Big Five / OCEAN)

### O - 开放性 (Openness to Experience)

| 语言信号 | 推断 |
|----------|------|
| 提到尝试新方法、学习新技能 | 高开放性 |
| 使用"新奇""创新""探索"等词 | 高开放性 |
| 强调"惯例""按规矩来""传统做法" | 低开放性 |
| 对比喻、抽象概念的接受度 | 高开放性标志 |

**工作关联:**
- 高开放性 → 创新岗位适配 ↑，但稳定重复性岗位适配 ↓
- 低开放性 → 适合流程化、合规要求高的岗位

---

### C - 尽责性 (Conscientiousness)

| 语言信号 | 推断 |
|----------|------|
| 频繁使用时间描述（具体日期、时间线） | 高尽责性 |
| 主动提到流程、规范、检查 | 高尽责性 |
| 对细节的具体描述 | 高尽责性 |
| 提到截止日期、承诺、责任 | 高尽责性 |
| 表述模糊（"差不多""大概其"） | 低尽责性 |
| 对过去的承诺轻描淡写 | 低尽责性 |

**工作关联:**
- 高尽责性 → 所有岗位的通用绩效预测因子（最强的单一预测指标，r≈.22-.31）
- 过度尽责 → 完美主义、灵活性不足

---

### E - 外向性 (Extraversion)

| 语言信号 | 推断 |
|----------|------|
| 高频使用社交词汇（"大家一起""团队"） | 高外向性 |
| 描述社交场景时情绪积极 | 高外向性 |
| 使用"安静""独立""专注"高频 | 低外向性（内倾） |
| 语速快、语句长、打断倾向 | 高外向性 |

**工作关联:**
- 高外向性 → 销售、管理、客户面向岗位适配 ↑
- 低外向性 → 研发、分析、独立工作适配 ↑
- 非判断性：内外向各有适配岗位

---

### A - 宜人性 (Agreeableness)

| 语言信号 | 推断 |
|----------|------|
| 高频使用合作性词汇（"我们""一起""帮助"） | 高宜人性 |
| 避免冲突性表述 | 高宜人性 |
| 对他人的负面评价直接且频繁 | 低宜人性 |
| 提到"公平""不公平"时反应强烈 | 高宜人性（正义敏感） |
| 用"他们""那些人"疏离地描述同事 | 低宜人性信号 |

**工作关联:**
- 高宜人性 → 团队协作、客户服务适配 ↑
- 低宜人性 → 可能在谈判、决策岗位有优势，但团队风险 ↑
- 极低宜人性 + 高外向性 → 暗黑三角风险 ↑

---

### N - 神经质 / 情绪稳定性 (Neuroticism)

| 语言信号 | 推断 |
|----------|------|
| 高频负面情感词 | 高神经质 / 低稳定性 |
| 过度担忧未来（"万一""如果...怎么办"） | 高神经质 |
| 对压力事件的强烈情绪反应 | 高神经质 |
| 描述挫折时使用灾难化语言 | 高神经质 |
| 情绪平稳，挫折描述冷静 | 低神经质 / 高稳定性 |
| 使用"没关系""可以处理"等韧性标记 | 高稳定性 |

**工作关联:**
- 高神经质 → 高压岗位风险 ↑，情绪劳动消耗大
- 低神经质 → 危机处理、高压环境适配 ↑
- 适度神经质 → 可能更细心、风险敏感

## 2. 暗黑三角 (Dark Triad)

### 自恋 (Narcissism)

| 语言信号 | 风险等级 |
|----------|----------|
| 高频自我指涉，过度使用"我"且内容自夸 | ⚠️ 中 |
| 夸大成就，缩小他人贡献 | ⚠️ 高 |
| 对批评/负面反馈的防御性反应 | ⚠️ 中 |
| 频繁对比自己与他人（"我比他们更..."） | ⚠️ 高 |
| 期待特殊待遇的潜台词 | ⚠️ 高 |
| "我带领团队实现了..."（忽略团队贡献） | ⚠️ 中 |

**工作风险:** 团队士气 ↓、人才流失 ↑、决策自我中心

---

### 马基雅维利主义 (Machiavellianism)

| 语言信号 | 风险等级 |
|----------|----------|
| 策略性语言（"策略""运作""布局"）异常高频 | ⚠️ 中 |
| 将人际关系描述为交易 | ⚠️ 高 |
| "必要之恶"合理化 | ⚠️ 高 |
| 对不同人描述同一事件时版本差异大 | ⚠️ 高 |
| 过度关注"信息差"和他人弱点 | ⚠️ 中 |
| 功利导向的成就观 | ⚠️ 中 |

**工作风险:** 政治斗争 ↑、信息操纵、信任破坏

---

### 精神病态倾向 (Psychopathy — 亚临床)

| 语言信号 | 风险等级 |
|----------|----------|
| 对他人痛苦的冷淡反应 | 🚨 高 |
| 缺乏悔意/责任承担的表述 | 🚨 高 |
| 冲动决策的叙事模式 | ⚠️ 中 |
| 冷漠或异常平淡的情感描述 | 🚨 高 |
| 刺激寻求的高频表述 | ⚠️ 中 |
| 将严重事件轻描淡写 | 🚨 高 |

**工作风险:** 安全风险 ↑、违规行为、职场霸凌

## 3. 归因风格 (Attributional Style)

> 基于 Weiner 归因理论 & Seligman 解释风格

### 三个维度

| 维度 | 积极归因（适应性） | 消极归因（风险信号） |
|------|-------------------|---------------------|
| **内部/外部** | 成功→内部（"我努力了"）；失败→平衡内外 | 成功→纯内部（自恋）；失败→纯外部（推责） |
| **稳定/不稳定** | 失败→不稳定（"这次没做好"） | 失败→稳定（"我永远做不好"） |
| **全局/特定** | 失败→特定于该情境 | 失败→泛化到所有事（"什么都做不好"） |

### 面试中的归因分析

```
问题: "为什么离开上一份工作？"

内控积极归因（健康）:
  "我在那里学到了很多，但发现自己的成长方向与公司战略不再匹配，
   所以主动寻找更契合的机会。"

外控消极归因（风险）:
  "领导不行，同事拖后腿，公司文化有问题，待不下去了。"
```

## 4. 动机理论 — 自我决定理论 (SDT)

> Deci & Ryan, 1985-2000

### 三种基本心理需求

| 需求 | 语言信号（满足） | 语言信号（未满足） |
|------|-----------------|-------------------|
| **自主性** (Autonomy) | "我决定...""我选择..." | "被迫...""没办法..." |
| **胜任感** (Competence) | "我有能力...""我擅长..." | "我不太会...""做不好" |
| **归属感** (Relatedness) | "团队很支持...""我们..." | "被孤立...""没人理解" |

### 动机类型连续体

```
外在动机 ←──────────────────────→ 内在动机

外部调节      内摄调节      认同调节      整合调节      内在动机
(为奖励/惩罚) (为自尊)      (认同价值)    (融入自我)    (纯粹兴趣)

风险递增 ←──────────────────────────────────── 稳定性递增 →
```

| 动机类型 | 语言标志 | 离职风险 | 绩效预期 |
|----------|----------|----------|----------|
| 外部调节 | "为了工资""没办法" | 高 | 仅满足最低要求 |
| 内摄调节 | "不能让人失望""要证明自己" | 中高 | 波动大 |
| 认同调节 | "我认为这个方向有意义" | 中 | 稳定 |
| 整合调节 | "这就是我想做的" | 低 | 高 |
| 内在动机 | "我就是喜欢做这个" | 最低 | 最高（创造力）|

## 5. 成就动机理论 (McClelland)

### 三种核心需求

| 需求类型 | 高需求者语言特征 | 适配岗位 | 风险 |
|----------|-----------------|----------|------|
| **成就需求** (nAch) | "目标""达成""突破""挑战" | 创业、项目负责人 | 过度竞争、忽视团队 |
| **权力需求** (nPow) | "影响""主导""控制""决策" | 高管、领导者 | 专权、压制他人 |
| **归属需求** (nAff) | "关系""和谐""团队""支持" | 协调、客服、HR | 回避冲突、难以做硬决定 |

### 需求组合分析

- 高 nAch + 低 nAff → 结果导向强，但可能人际关系粗糙
- 高 nPow + 低 nAff → 领导力风险（可能专制）
- 高 nPow + 高 nAch + 低 nAff → 暗黑三角风险 ↑
- 高 nAff + 低 nAch → "老好人"，可能绩效平庸

## 6. 心理资本 (PsyCap) — Luthans

| 维度 | 语言信号 | 反向信号 |
|------|----------|----------|
| **自我效能** | "我可以""我有经验处理" | "我不确定能不能" |
| **希望** | "有办法""多条路" | "没有出路""只能这样" |
| **韧性** | "之前也遇到过，挺过来了" | "这次不一样，太难了" |
| **乐观** | 积极未来导向表述 | 消极未来预期 |
