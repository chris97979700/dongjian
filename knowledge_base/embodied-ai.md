## 2. 强化学习 (Reinforcement Learning)

### 核心范式
| 方法 | 原理 | 应用 |
|------|------|------|
| **PPO** (Proximal Policy Optimization) | 策略梯度，限制更新幅度 | 最广泛的机器人RL算法 |
| **SAC** (Soft Actor-Critic) | 最大熵RL，探索能力强 | 灵巧手操作 |
| **Q-Learning / DQN** | 值函数方法 | 离散动作空间 |
| **Imitation Learning** | 从人类示范学习 | 数据采集 + RL微调 |
| **Sim-to-Real Transfer** | 仿真训练→现实部署 | 域随机化/域适应 |

### 面试信号识别
| 候选人表述 | 推断 |
|-----------|------|
| "我做过sim-to-real transfer" | 有完整的RL工程经验 |
| "调过PPO/SAC在机械臂上" | 有实际部署经验（稀缺） |
| "用behavior cloning预训练+RL微调" | 理解IL+RL组合策略 |
| "域随机化解决sim-to-real gap" | 了解前沿技术细节 |

### 关键挑战
- **样本效率**: 真实机器人数据采集成本极高
- **奖励设计**: Reward engineering仍是核心瓶颈
- **泛化能力**: 单一任务→多任务的跨越

## 3. 仿真 (Simulation)

### 主流仿真平台
| 平台 | 开发者 | 特点 | 适用场景 |
|------|--------|------|---------|
| **Isaac Sim** | NVIDIA | GPU加速、物理精度高 | 工业级仿真 |
| **MuJoCo** | Google DeepMind | 轻量、快速 | 学术研究首选 |
| **PyBullet** | 开源 | 易上手 | 原型验证 |
| **Isaac Gym** | NVIDIA | 大规模并行训练 | RL训练加速 |
| **SAPIEN** | UCSD | 关节物体交互 | 操作任务 |
| **Genesis** | 北大/清华等 | 生成式物理引擎 | 前沿研究 |

### 仿真关键能力
- **域随机化** (Domain Randomization): 材质/光照/摩擦随机化训练→提高真实泛化
- **并行训练**: Isaac Gym 万级环境并行，训练从几天→几小时
- **数字孪生**: 真实工厂/仓库的高精度数字复刻

## 4. 世界模型 (World Models)

### 核心概念
让机器人"想象"行动的后果，在"脑内"推演，减少真实试错。

### 代表工作
| 方法 | 机构 | 思路 |
|------|------|------|
| **DreamerV3** | DeepMind | 学习世界模型 + 在想象中训练策略 |
| **TD-MPC2** | UC Berkeley | 模型预测控制 + RL |
| **UniSim** | Google DeepMind | 通用视频生成世界模型 |
| **Sora/Video Generation** | OpenAI/Google | 视频生成作为世界模拟器 |
| **GR-1/GR-2** | 字节跳动 | 视频预训练 + 机器人微调 |

### 行业意义
- 减少真实机器人数据需求（降低90%+）
- 实现few-shot/zero-shot任务泛化
- 2025年起从学术走向工业落地

## 5. 操作 (Manipulation)

### 技术栈
| 层次 | 技术 | 说明 |
|------|------|------|
| 感知 | 6D姿态估计、点云配准、触觉传感 | 知道物体在哪、什么形状 |
| 规划 | 运动规划(Motion Planning)、抓取规划(Grasp Planning) | 怎么拿 |
| 控制 | 阻抗控制、导纳控制、力位混合控制 | 精确执行 |

### 前沿方向
- **灵巧手**: Shadow Hand, Leap Hand, 因时机器人
- **双手协作**: ALOHA, Mobile ALOHA 系列
- **触觉反馈**: GelSight, DIGIT 触觉传感器
- **通用操作**: RT-2 (Google), Octo (Berkeley), π0 (Physical Intelligence)

### 面试评估维度
| 经验类型 | 价值判断 |
|---------|---------|
| 接触过真实机械臂（UR/KUKA/Franka） | 高价值 — 理解真实部署挑战 |
| 只在仿真中做操作 | 中等 — 需了解sim-to-real经验 |
| 做过force/torque控制 | 高价值 — 精密操作核心能力 |
| 有双手/灵巧手经验 | 稀缺人才 |

## 6. 导航 (Navigation)

### 技术栈
| 层次 | 技术 |
|------|------|
| SLAM | ORB-SLAM3, RTAB-Map, 激光/视觉SLAM |
| 定位 | AMCL, 卡尔曼滤波, 视觉定位 |
| 路径规划 | A*, RRT*, 全局/局部规划 |
| 决策 | 行为树, 有限状态机, RL导航 |

### 前沿方向
- **视觉语言导航(VLN)**: 自然语言指令→导航（"去厨房拿杯子"）
- **NeRF/3D Gaussian Splatting**: 神经场景表示用于导航
- **社交导航**: 人群中的机器人导航（考虑行人意图）
- **端到端导航**: ViNT (Berkeley), NoMaD

### 行业应用
| 场景 | 代表公司 |
|------|---------|
| 仓储物流 | Amazon Proteus, 极智嘉, 海柔创新 |
| 酒店/清洁 | Relay, 高仙机器人 |
| 最后一公里配送 | Starship, Nuro |
| 家庭服务 | 扫地机器人（石头/科沃斯/追觅） |

## 7. 电机与驱动 (Actuation)

### 电机类型
| 类型 | 特点 | 应用 |
|------|------|------|
| **无刷直流电机(BLDC)** | 高效率、高转速 | 四足/人形关节 |
| **伺服电机** | 精确位置控制 | 工业机械臂 |
| **步进电机** | 开环控制、低成本 | 3D打印机、小负载 |
| **直驱电机** | 无减速器、高动态 | 高性能关节 |
| **直线电机** | 直接线性运动 | 精密定位平台 |

### 核心指标
- **扭矩密度**: Nm/kg，决定关节性能
- **控制带宽**: 高频响应能力
- **反向驱动性**: 安全性关键
- **编码器精度**: 位置控制精度

### 减速器
| 类型 | 特点 |
|------|------|
| **谐波减速器** | 高精度、零背隙、贵（日本Harmonic Drive垄断） |
| **RV减速器** | 高刚性、大扭矩、工业机器人主流 |
| **行星减速器** | 中等精度、成本适中 |
| **摆线减速器** | 类RV，国产替代方向 |

## 9. 行业趋势与人才判断

### 2025-2026 关键趋势
1. **从"单任务"到"通用"**: 通用机器人基础模型兴起（π0, RT-2, Octo）
2. **仿真驱动开发**: Isaac Sim等工业化仿真工具链成熟
3. **硬件成本快速下降**: 人形BOM成本从百万→<30万
4. **数据飞轮**: 更多部署=更多数据=更强模型
5. **中国供应链优势**: 电机、减速器、传感器国产化加速
6. **软件定义硬件**: 传统机电→AI驱动的范式转换

### 候选人技术背景评估矩阵
| 背景 | 岗位匹配 | 稀缺度 |
|------|---------|--------|
| RL + 真实机器人部署 | 操作/控制核心岗 | ★★★★★ |
| 纯仿真经验 | 仿真/算法岗 | ★★★ |
| CV + 机器人 | 感知/抓取岗 | ★★★★ |
| SLAM + 自动驾驶 | 导航岗 | ★★★ |
| 电机/控制（无AI） | 硬件/底层岗 | ★★★ |
| 基础模型 + 机器人 | 前沿研究岗 | ★★★★★ |

### 面试追问方向
- "你用过的仿真平台是什么？解决了什么sim-to-real问题？"
- "你做的RL是在仿真还是真机上训练的？数据量多大？"
- "对当前的人形机器人行业怎么看？你觉得最大的技术瓶颈在哪？"
- "如果有无限预算，你会怎么设计一个通用操作机器人？"
