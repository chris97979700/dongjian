## 语言指纹

**人称代词:**
- 高频"我" → 自我关注/焦虑 · 低频"我" → 回避责任
- 高频"我们" → 团队归属 · 低频"我们" → 疏离
- "我"突然变"我们" → 责任转移信号
- 高频"你" → 防御/指责

**动词时态:**
- 过去时主导 → 反思型 · 将来时主导 → 规划型
- 条件句增多("如果能...") → 不满或离职倾向

**认知与情感词:**
- 高频认知词("认为/分析/逻辑") → 理性主导
- 高频情感词 → 情绪驱动 · 积极>消极 → 心理韧性
- 绝对词("一定/绝对/永远") → 僵化思维/隐瞒
- 模糊词("大概/也许/差不多") → 回避精确性

**填充词 & 停顿:**
- 高频"嗯/那个" → 中文口语习惯，不可过度解读
- 提问后停顿过长 + 填充词骤增 → 可能编造

## 行为模式

**大五人格 (OCEAN):**

O-开放性: 提到"新奇/创新/探索"→高 · 强调"惯例/规矩"→低
C-尽责性: 具体时间/流程/承诺→高 · 模糊应付→低
E-外向性: 主动扩展/社交→高 · 简洁被动→低
A-宜人性: 共情/合作→高 · 竞争/质疑→低
N-神经质: 反复担忧/情绪词多→高 · 稳定平静→低

**暗黑三角:**
- 自恋: 频繁自我指涉、夸大成就、需要崇拜
- 马基雅维利: 策略性语言、人脉=工具、结果导向不择手段（注：谈判中的策略行为≠马基雅维利）
- 精神病态: 对他人痛苦漠然、冲动冒险描述

**归因风格:**
- 内控+稳定: "我擅长这个" · 外控+不稳定: "运气问题"
- 成功内归因+失败外归因 → 自利偏差

## 风险信号

**离职风险:**
- 时间导向变化: "我们明年"→"公司明年"（心理距离 ↑）
- 对比性语言增多: "这里 vs 外面"
- 条件句: "如果留在这里..."
- 不再用将来时讨论团队 → 即将离开

**诚信风险:**
- 故事过于完美无细节 → 可能编造
- 回避具体问题、转移话题
- 人称代词突变（"我"→"我们"）
- 主动提供过多无关细节 → 过度补偿

**稳定性风险:**
- 频繁跳槽合理化("每次都是公司的错")
- 对过去雇主的极端评价
- 情绪稳定性差（高频情感波动词汇）

**适配风险:**
- 价值观冲突信号: 对团队文化露出轻视
- 期望不匹配: 对岗位理解偏差大
- 工作风格错配: 自主需求 vs 结构化需求

## 情景逻辑

**面试场景判定:**
- 初次面试 vs 终面: 前者探索性语言多，后者承诺性语言多 → 不同阶段相同语言含义不同
- 主动求职 vs 被动挖猎: 前者"寻求"框架，后者"被认可"框架
- 有竞品Offer vs 无Offer: 前者语言中锚定和比较增多

**文化语境修正:**
- 中文高语境文化: 人称省略、"嗯/那个"不可过度解读
- 谦虚表达("还行/一般")可能是礼貌而非低自信
- 间接拒绝("我再想想"≈拒绝) vs 欧美直接拒绝

**关键误区:**
- 策略性谈判 ≠ 马基雅维利人格
- 单一对话片段信息有限 → 降低对应维度置信度
- 不同文化背景的沟通风格差异不是人格缺陷

## 公司软性素质

六维度评估框架:

| 维度 | 高信号 | 低信号 |
|------|--------|--------|
| 实事求是 | 数据支撑、承认不足、区分事实与观点 | 笼统断言、回避细节、过度包装 |
| 追求极致 | 对细节有要求、迭代改进的叙述 | "差不多就行"、缺乏质量标准 |
| 思考力 | 深层追问、系统框架、第一性原理 | 表面回答、缺乏逻辑链 |
| 持续学习 | 成长叙事、跨界学习、反思失败 | 停滞叙述、归因外部 |
| 会抓重点 | 能区分主次、快速定位核心问题 | 细节淹没重点、平均用力 |
| 技术品味 | 对行业/技术有独立判断和审美 | 跟随式、缺乏观点 |

## 自校准原则

1. 每个判断标注置信度（高/中/低），低置信度维度需明确说明
2. 标注信息不足的盲区，主动建议追问方向
3. 中文填充词（嗯、那个、就是说）不可作为主要判断依据
4. 分析基于统计学模式，提供参考而非绝对判断

## 谈判策略

**质疑分类:**
- 薪资: 锚定对抗、结构质疑(期权价值)、公平性(社会比较)、时机施压(多Offer)、反向探底
- 职业: 成长焦虑、平台比较、技能贬值、角色模糊
- 公司: 融资焦虑、团队规模、方向PMF、竞争碾压
- 文化: 加班文化、管理风格、价值观冲突

**策略框架:**
- 硬策略: 直接锚定、施压、制造紧迫感 → 适合有信息优势时
- 软策略: 共情、关系建设、情感绑定 → 适合候选人安全感低时
- 原则策略: 拆解框架、价值共创、重新定义问题 → 适合对方设置了陷阱框架时

**认知武器（慎用）:**
- 锚定效应: 先出价者设锚
- 社会认同: 引用同行/同级别案例
- 稀缺性: 限时/限量/独家机会
- 一致性: 用候选人之前说过的话回引
- 互惠: 先给再要

**关键原则:**
- 对人温和，对事强硬
- 永远不要在他的棋盘上下棋
- 薪资谈判核心: 把单一数字扩展为多维价值（现金+期权+成长+影响力+团队）
- 谈判中的策略性行为 ≠ 马基雅维利人格

## 沟通经典分析框架

**非暴力沟通 (Rosenberg):**
- 观察vs评价: 说事实("迟到3次")=高尽责, 贴标签("不靠谱")=低宜人
- 感受命名精度: 精确("焦虑/沮丧")=高情绪智力, 笼统("不爽")=低
- 暴力语言: 指责/比较/强求("必须/应该") → 暗黑三角↑
- 非暴力: "我感到...因为我需要..." → 宜人↑内控↑

**关键对话 (Patterson):**
- STATE: 事实→故事→询问→试探→鼓励 = 高沟通力
- 压力下沉默或暴力 → 低情绪稳定性
- "我不希望...但是我想..." → 高情商冲突处理

**批判思维 (Browne & Keeley):**
- 结论+理由链是否清晰 → 思考力
- 谬误识别: 虚假两难("要么80万要么不重视")、稻草人、灾难化
- 价值观假设: 候选人认为什么最重要(钱/成长/安全)

**金字塔原理 (Minto):**
- SCQA结构(情境-冲突-问题-答案) → 会抓重点
- MECE原则(不重叠不遗漏) → 系统思考力
- 演绎vs归纳偏好的 → 逻辑型vs模式型

**毛选辩证分析:**
- 主要矛盾 vs 次要矛盾: 候选人核心诉求是什么, 不被噪音干扰
- 透过现象看本质: "要高薪"是现象, 本质可能是"需要价值认可"
- 实事求是: 用证据说话, 不凭感觉判断
- 一分为二: 不极端化, 每个候选人都有优点和风险
- 发展的观点: 现在不等于未来, 动态评估
- 抓住主要矛盾: 面试中识别最关键信号
- 务实vs空谈: "我做了什么" > "我认为应该..."
- 批评与自我批评: 能否客观说自己的不足 → 宜人↑内控↑

## 劳动法（辞退谈判）

**解除类型:**
- 协商解除(36条): 最低风险，双方协商补偿
- 过失性(39条): 0补偿，需充分证据（违纪/失职/欺诈等）
- 无过失性(40条): N+1，适用不胜任/医疗期满/客观情况变化
- 经济裁员(41条): N，需法定程序

**补偿计算:**
- N = 前12月均工资 × 工作年限（满1年=1月，6月+=1年，<6月=0.5月）
- N+1: 第40条解除 + 代通知金
- 2N: 违法解除（无正当理由/程序违法/证据不足）

**谈判原则:**
- 以法为基: 所有方案必须合法，不给法外承诺
- 以人为本: 体面分手，承认贡献
- 利益导向: 从"谁对谁错" → "如何体面结束"
- 协议必备: 解除日期、补偿金额、工资结算、双方再无争议条款

**禁区:**
- 不得威胁/恐吓/欺诈逼迫签字
- 三期女职工原则上不得依据40/41条解除
- 工伤1-4级不得解除
- 竞业限制补偿 ≥ 30%月均工资
