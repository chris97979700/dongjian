# 谈判策略引擎 — 候选人质疑应对

> 基于谈判心理学（Cialdini影响力、Fisher & Ury原则谈判、Malhotra谈判天才）、认知偏差、说服理论
> 用途: 输入候选人问题/质疑，输出结构化的谈判策略

---

## 1. 谈判策略工作流

```
候选人质疑输入
    │
    ▼
┌──────────────────┐
│ 1. 深层需求解码    │  ← 这句话表面在问什么？实际在担心什么？
├──────────────────┤
│ 2. 策略选项生成    │  ← 3种不同策略（硬/软/原则）
├──────────────────┤
│ 3. 推荐方案        │  ← 基于候选人画像的最佳策略
├──────────────────┤
│ 4. 话术参考        │  ← 可直接使用的措辞
├──────────────────┤
│ 5. 风险提示        │  ← 该策略可能的后坐力
└──────────────────┘
```

## 2. 质疑分类体系

### A. 薪资类质疑

| 质疑类型 | 示例 | 深层需求 |
|----------|------|----------|
| 锚定对抗 | "DJI 能给到80万" | 期望你来匹配，或测试你的薪资天花板 |
| 结构质疑 | "你们期权值多少？" | 不确定非现金部分的价值，需要可量化参照 |
| 公平性质疑 | "我同学同年都拿这个数" | 社会比较，需要被认可价值 |
| 时机施压 | "我这边还有两个offer在等回复" | 制造紧迫感，压缩你的决策时间 |
| 反向探底 | "你们的 budget 大概多少？" | 探你的底线，避免自己先出价 |

### B. 职业发展质疑

| 质疑类型 | 示例 | 深层需求 |
|----------|------|----------|
| 成长焦虑 | "一年后我能做什么？" | 担心停滞，需要可预见的晋升路径 |
| 平台比较 | "大厂有更好的背书" | 职业安全感，担心创业经历贬值 |
| 技能贬值 | "这个技术方向会不会过时？" | 职业可持续性焦虑 |
| 角色模糊 | "具体负责什么？title是什么？" | 需要确认scope和影响力 |

### C. 公司风险质疑

| 质疑类型 | 示例 | 深层需求 |
|----------|------|----------|
| 融资焦虑 | "你们烧到什么时候？" | 担心公司倒闭 + 自己裸辞风险 |
| 团队质疑 | "团队才10个人？" | 怀疑执行力和资源 |
| 方向质疑 | "这个市场真的有需求吗？" | 对PMF的不确定 |
| 竞争焦虑 | "大厂也做这个怎么办？" | 担心被巨头碾压 |

### D. 文化/适配质疑

| 质疑类型 | 示例 | 深层需求 |
|----------|------|----------|
| 稳定性焦虑 | "创业公司加班会很凶吧" | 不想用健康换钱 |
| 管理适配 | "扁平管理真的管用吗？" | 担心混乱，需要结构 |
| 氛围担忧 | "团队气氛怎么样？" | 上一家可能有创伤经历 |

## 3. 策略工具箱

### 策略一：硬策略（对抗/锚定）⚠️ 高风险

| 适用场景 | 方法 | 风险 |
|----------|------|------|
| 候选人明显在压价 | 直接锚回去："我们的offer如果低于X，你不会考虑？" | 可能激怒 |
| 候选人已有更强offer | 承认差距，强调差异化 | 可能被认为是放弃 |
| 候选人不合理要求 | 坦诚上限："这是我们的top of band" | 谈判可能终止 |

### 策略二：软策略（关系/情感）🟢 低风险

| 适用场景 | 方法 |
|----------|------|
| 候选人犹豫 | "我不着急，你好好想想——选对方向比选对薪资重要"|
| 候选人表达顾虑 | 先共情再回应："我理解你的担心，换我也会想..." |
| 建立信任 | 提供额外信息（非机密）展示诚意 |

### 策略三：原则策略（价值共创）🟢 推荐

| 适用场景 | 方法 |
|----------|------|
| 所有场景 | 基于客观标准谈判，寻找双方利益交集 |
| 薪资分歧 | "我们一起算算这个package对你意味着什么" |
| 方向分歧 | "如果我告诉你明年的路线图，你会怎么想？" |

## 4. 认知武器库（善用但谨慎）

| 武器 | 方法 | 适用场景 |
|------|------|----------|
| **锚定效应** | 先出价者定基调，但不要先出价 | 候选人问"budget多少" |
| **稀缺感** | "这个role我们一年只开一两个" | 候选人犹豫 |
| **社会认同** | "XX（候选人认识的）也选择加入了我们" | 候选人担心风险 |
| **一致性** | 回顾候选人之前说过的"你最看重方向" | 候选人被薪资带偏 |
| **损失厌恶** | "如果你不来，这个方向就少了一个关键的人" | 最终push |
| **互惠** | 主动让步（不损害底线的让步） | 打破僵局 |

> ⚠️ 认知武器是双刃剑。候选人通常也懂这些，一旦被感知为"套路"则信任崩塌。

## 5. 原则谈判框架（Fisher & Ury）

### 四项基本原则

1. **对人温和，对事强硬**
   - "我很欣赏你的坦诚"（对人）+ "但这个数字确实超出了我们的band"（对事）

2. **聚焦利益，而非立场**
   - 候选人的立场："我要50%涨幅"
   - 候选人的利益："我需要被认可价值 + 财务安全感"
   - 你的回应：不直接对抗50%，而是在利益层面满足

3. **创造双赢选项**
   - "薪资我们卡在X，但如果加上期权价值和年底review，12个月后你在乎的总数其实是..."

4. **坚持客观标准**
   - 引用行业数据："同阶段公司同岗位的50分位是...75分位是..."

## 6. 多Offer场景策略

### 候选人有竞品offer的典型心态

```
                   竞品Offer
                      │
          ┌───────────┼───────────┐
          ▼           ▼           ▼
      "你匹配它"  "用它压你"  "我真纠结"
     (价格战)   (谈判武器)  (真实困境)
```

| 候选人心态 | 识别信号 | 策略 |
|-----------|----------|------|
| 价格战 | "他们给了X，你们呢？" 语气直接 | 差异化，不打价格战 |
| 谈判武器 | 主动提起但不过分强调 | 确认真实性，不急于跟进 |
| 真实困境 | 语气犹豫，主动分析利弊 | 帮ta梳理决策维度，赢得信任 |

### 回应框架

```
当候选人说"DJI给了更高的offer"

❌ 错误: "我们也可以match"（价格战，两败俱伤）
❌ 错误: "DJI加班更狠"（攻击竞品，显得小气）

✅ 方案1（差异法）:
  "DJI是一家伟大的公司。但我很好奇——
   你在DJI能接触到的决策层和影响力，和我们这里相比，
   哪边一年后对你职业生涯的杠杆更大？"

✅ 方案2（时间法）:
  "不用现在做决定。你可以两边都走完流程。
   最后选的不是offer，是未来两年你真正想干什么。"

✅ 方案3（现实验证法）:
  "大家都是未上市的期权。我们红杉+凯龙背书，
   首款产品今年发，你觉得哪边的期权更可能变现？"
```

## 7. 薪资谈判专项

### 候选人报了一个很高的期望 → 三步回应

```
第一步: 先不回应数字，先挖掘
  "这个数字背后，你最看重的其实是哪部分？现金？期权？还是未来涨幅？"

第二步: 分层拆解
  "如果我把这个数拆开——现金、期权、年终、签字费——你心里各是多少？"

第三步: 差异化满足
  "现金我们卡在X，但期权和签字费我们有弹性空间。这样重新算下来..."
```

### 薪资谈不拢时的备用筹码

| 筹码 | 价值 | 适用对象 |
|------|------|----------|
| 签约奖金 | 一次性的满足 | 短期现金流敏感的 |
| 加速行权 | 长期价值 | 看重股权但犹豫的 |
| 早期review承诺 | 6个月后调整薪资 | 对自己有信心但希望有保障的 |
| title upgrade | 面子+市场价值 | 职业阶段敏感的 |
| 远程/弹性 | 生活质量 | 有两地需求的 |
| 教育预算/参会 | 成长投资 | 技术驱动型 |

## 8. 时间线策略

| 阶段 | 策略 |
|------|------|
| 初次接触 | 以了解为主，不暴露薪资范围 |
| 二面后 | 开始试探期望，但不承诺 |
| 终面前 | 确认双方差距，准备筹码方案 |
| offer时 | 给一个"温和的意外"——比暗示的多出一点 |
| 犹豫期 | 48小时法则：给两天，不过度push |
| 拒绝时 | 优雅退场："以后有缘分再合作"——候选人会记得 |

---

> 谈判不是把对方打趴下。最好的谈判是双方都觉得自己赚了。
