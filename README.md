# 洞见 DongJian — 人才风险分析

基于心理学文献的 AI 人才分析工具。

**功能：**
- 💬 **对话分析**：从面试对话中提取动机、人格、风险信号
- 🎯 **谈判策略**：针对候选人质疑生成应对策略

**配置：**
部署前在 HuggingFace Settings → Secrets 中设置：
- `LLM_API_KEY`：DeepSeek API Key
